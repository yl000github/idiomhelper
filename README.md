## 微信小程序成语消消看辅助


### 游戏简介
![image](https://github.com/yl000github/idiomhelper/blob/master/img/game.png)
  找出所有匹配，顺序点击，消完所有过关
  
### 工作原理

  最近看到跳一跳和答题的辅助，仿照写个类似的
1.   通过adb截图获得题目
2.   通过百度ocr识别获得文字信息
3.   从网上获取常用的成语数据库
4.   查找匹配成语，一一筛选，找出符合的成语

### 要求

1. 需百度ocr，网上注册一个 
2. 安装mysql数据库，导入成语库
3. 安装python依赖

### 运行过程
直接运行mobile_idiom.py，根据提示手动操作

### 遗留问题
1. ocr识别可能有误
2. 游戏官方的成语库与网上下载的不匹配，造成了识别出来的一些成语可能无法通过，所以偶尔需要人手工干预
  

有问题欢迎issue
